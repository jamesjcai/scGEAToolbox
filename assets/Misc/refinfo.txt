CellBender [PMID:37550580]	CellBender is a software package for eliminating technical artifacts, including ambient RNA contamination, from high-throughput single-cell RNA sequencing (scRNA-seq) data, improving data quality for downstream analyses.
Gene Program Correlation Matrix	A matrix depicting the pairwise correlation between gene program activities (co-variation scores) across samples, adjusted by permutation tests. This analysis identifies significantly co-regulated gene programs and densely interconnected modules among cell compartments. Results are visualized as heatmaps and network graphs, with statistical significance assessed via permutation of sample identity (FDR < 0.1).
Panhumanpy	Pan-Human Azimuth is a neural-network-based classifier that annotates human single-cell or single-nucleus RNA-seq data consistently across tissues and technologies. It covers over 380 cell types from 23 tissues, organized into a hierarchical cell ontology based on an adapted version of the DISCO reference. [https://satijalab.org/pan_human_azimuth/]
SGS Analysis [PMID:39828795]	Stochastic Gene Silencing (SGS) analysis leverages natural variability in single-cell gene expression to assess gene function. For a target gene, cells are classified as transcriptionally active (g+) or silenced (g−). Comparing these groups identifies differentially expressed (SGS-responsive) genes that reveal the functional role of the target gene.
Single Click Solution	Single Click Solution (from raw data to annotation) will embed, cluster and annotate cells automatically.

Sceodesic	Sceodesic is a Python package implementing the gene program discovery algorithm by Ozbay, Parekh, and Singh. Given a single-cell gene expression dataset, it discovers meaningful gene programs and re-interprets gene expression as activity levels of those programs.
SCORPION [PMID:38438786]	SCORPION uses a message-passing algorithm to reconstruct comparable gene regulatory networks from single-cell/nuclei RNA-seq data. The resulting networks are suitable for population-level comparisons and leverage shared baseline priors across samples.
SCimilarity [PMID:39566551]	SCimilarity is a unified representation of single-cell expression profiles that quantifies similarity between expression states and generalizes to new studies without additional training. It enables cell search across millions of profiles to find cells similar to a query state, helping researchers leverage large public scRNA-seq atlases.
DE in Batch Mode	This function iterates over all cell types. For each cell type, DE analysis based on the Wilcoxon rank-sum test compares two selected cell groups (e.g., WT and KO). Results are saved as Excel files in a user-specified folder.
DP in Batch Mode	This function iterates over all cell types. For each cell type, DP analysis compares two selected cell groups (e.g., WT and KO). Results are saved as Excel files in a user-specified folder.
DV in Batch Mode	This function iterates over all cell types. For each cell type, DV analysis compares two selected cell groups (e.g., WT and KO). Results are saved as Excel files in a user-specified folder.
DE Analysis	Differential Expression (DE) Analysis identifies genes with significant differences in mean expression between two conditions or cell groups.
DP Analysis	Differential Program (DP) Analysis [PMID:40113778] identifies gene expression programs — sets of functionally related genes — expressed at varying levels between conditions, providing biological insights into processes affected by the conditions of interest.
DV Analysis	Differential Variability (DV) Analysis identifies genes with significant differences in expression variability between groups, detected using the Spline-DV method.
Geometric Sketching [PMID:31176620]	Geometric sketching accelerates single-cell RNA-seq analysis by selecting a small, representative subset of cells that captures transcriptomic diversity. Sketches provide more comprehensive visualization, improve detection of rare cell types, and make very large datasets tractable for downstream analysis.
MELD [PMID:33558698]	MELD estimates, for every cell in a dataset, the likelihood of belonging to each experimental sample. By comparing per-sample density estimates, MELD quantifies the effect of a perturbation at the single-cell level — without requiring prior clustering — identifying cells most or least affected by the treatment.
DecontX [PMID:32138770]	DecontX is a Bayesian method that estimates and removes ambient RNA contamination in individual cells from droplet-based scRNA-seq data. Contaminating transcripts from other cell populations are identified and corrected, improving the accuracy of downstream cell type and expression analyses.
Harmony [PMID:31740819]	Harmony integrates single-cell RNA-seq datasets from different technologies or conditions by projecting cells into a shared embedding where they group by cell type rather than batch. It simultaneously corrects for multiple experimental and biological confounders, enabling robust cross-dataset analysis.
Scrublet [PMID:30954476]	Scrublet detects cell multiplets in scRNA-seq data — barcodes labeling more than one cell — and estimates their effect on downstream analyses. Each cell is scored for multiplet likelihood, helping users identify and filter problematic barcodes before further analysis.
scTenifoldNet [PMID:33336197]	scTenifoldNet constructs and compares single-cell gene regulatory networks (scGRNs) using principal-component regression, tensor approximation, and manifold alignment. It reveals regulatory changes between conditions by comparing scGRNs, identifying gene programs associated with specific biological processes.
scTenifoldKnk [PMID:35510185]	scTenifoldKnk performs virtual gene knockout (KO) analysis using scRNA-seq data from wild-type samples. A gene regulatory network is constructed, the target gene is removed in silico, and manifold alignment identifies differentially regulated genes — inferring gene function without requiring actual KO experiments.
GenKI [PMID:37246643]	GenKI predicts gene function via virtual knockout using only wild-type scRNA-seq data. A variational graph autoencoder learns the gene regulatory network; the target gene is then knocked out in silico. Differences between wild-type and virtual KO latent representations reveal the gene's cell type-specific functional role.
scTenifoldXct [PMID:36787742]	scTenifoldXct detects ligand-receptor-mediated cell-cell interactions by aligning gene regulatory networks across interacting cell populations. It maps cellular communication graphs and can compare interactions between conditions (e.g., healthy vs. diseased) to identify differential intercellular signaling.
GCL Analysis [PMID:33139959]	GCL (Gene Co-expression Level) analysis measures the average multivariate dependency between randomly chosen gene subsets across single cells. Genes are partitioned into two random subsets, and the dependency between subsets is computed across a cohort of cells to quantify global gene co-expression.
talklr [DOI:10.1101/2020.02.01.930602]	talklr is an information theory-based approach to uncover potential ligand-receptor interactions involved in tissue homeostasis and disease. Unlike methods that analyze each gene-cell type pair separately, talklr simultaneously considers expression changes in both ligands and receptors across multiple cell types and conditions.
PanglaoDB [PMID:30951143]	PanglaoDB is a manually curated database of cell type markers compiled from thousands of published articles. Cell clusters are annotated by scoring their gene expression against known marker sets, enabling automated cell type assignment for scRNA-seq cluster analysis.
SnnDpc [DOI:10.1016/j.ins.2018.03.031]	SNN-DPC is an improved density-peaks clustering algorithm that redefines local density and inter-point distances using shared nearest neighbor information. This makes it more robust to variable-density clusters compared to the original DPC algorithm.
Differentiation Potency [PMID:33244588]	Differentiation potency estimation identifies stem and multipotent cells in scRNA-seq data without requiring temporal information or known hierarchies. The CCAT (Correlation of Connectome and Transcriptome) measure returns accurate potency estimates for millions of cells within minutes.
Predefined Cell Score	This function calculates a gene signature score for a user-defined gene set. Scores are computed using either the UCell method (default) — which is rank-based and robust across datasets — or Seurat's AddModuleScore, which normalizes by binning genes of similar expression at the population level.
HVG Functional Analysis [PMID:31861624]	This function quantifies single-cell expression variability (scEV) within a homogeneous group of cells. Results support the "variation is function" hypothesis, indicating that cell type-specific, higher-level system functions depend on quantifiable expression variability.
SPIA [PMID:18990722]	SPIA (Signaling Pathway Impact Analysis) combines classical gene set enrichment with a novel perturbation measure that evaluates the actual impact on pathway topology under a given condition. This provides a more informative assessment of pathway activity changes than enrichment analysis alone.
scTenifoldCko [Unpublished]	scTenifoldCko performs virtual knockout experiments to study cell-cell communication using scRNA-seq data. By computationally removing genes involved in ligand-receptor interactions, it identifies critical mediators of intercellular signaling and assesses their roles in biological processes or disease, without requiring actual knockout experiments.
